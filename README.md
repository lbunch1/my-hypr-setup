# My Hyprland Setup

## Installation

Clone this repo, cd into the cloned directory and run

```
./setup.sh
```

## Hyprland Shortcuts

| Action | Keystroke |
|--------|-----------|
| App Launcher | Meta - Space |
| Kitty | Meta - a |
| Brave | Meta - w |
| File Browser | Meta - n |
| Close Window | Meta - q |
| Maximize Window | Meta - Enter |
| Bluetooth Control | Meta - b |
| Switch to workspace # | Meta - # |
| Move Window to workspace # | Meta - Shift - # |
